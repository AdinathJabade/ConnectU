# ConnectU – College Community Platform (MERN + Socket.IO)

A real-time college network to connect students, alumni, teachers, and TPOs for career and placement discussions.

## Project Structure

- Frontend: React.js (Vite) + Tailwind CSS
- Backend: Node.js + Express + MongoDB
- Realtime Messaging: Socket.IO

## Features

- Role-based dashboards (Student, Alumni, Teacher, TPO)
- Real-time chat (Student ↔ Alumni)
- Gamified Q&A system
- Notice board from TPOs/Teachers
- Alumni job post verification by TPO
